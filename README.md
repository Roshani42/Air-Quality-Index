# BreatheSafe 🌬️

A real-time air quality dashboard built with pure HTML, CSS, and JavaScript — no frameworks.

Search any city and instantly see the AQI, pollutant breakdown, 48-hour forecast, and health advice based on current air conditions.

---

## Features
- Real AQI from WAQI ground sensors (matches Google & AQI.in)
- Pollutant breakdown — PM2.5, PM10, O₃, NO₂, SO₂, CO
- 48-hour forecast chart (Canvas)
- Health advice based on AQI level
- My Location (geolocation)
- Recent searches, dark/light mode
- Source + timestamp on every reading

## Tech
HTML · CSS · JavaScript · WAQI API · OpenWeatherMap API
